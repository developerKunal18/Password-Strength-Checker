#  Password Strength Checker

## Overview
The **Password Strength Checker** is a simple Python program that evaluates the strength of a password using basic security rules.

It helps users understand whether their password is weak, medium, or strong.

---

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Simple strength classification

---

## Technologies Used
- Python
- string module
- Conditional logic
- Loops
- User input handling

---

## How It Works
1. User enters a password  
2. Program checks multiple strength rules  
3. Each rule contributes to a score  
4. Password strength is displayed
